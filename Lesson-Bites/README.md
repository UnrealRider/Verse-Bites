## 初步知道 Verse 的优势与问题
### 优势
- 大幅降低 UEFN Device 的使用来优化与提升岛屿的内存使用
- 实现与突破 UEFN Device 的制作瓶颈
- 实现 UEFN 快速的编辑与测试

### 问题
- 实际上 Verse 的许多功能实现是被 UEFN Device 所限制的
- Verse 还有很多功能没有被开放
- 总结：Verse 没有你想的自由
