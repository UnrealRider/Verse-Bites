## 学习 Verse 前的准备
- 🎥 [Coming Soon...](https://github.com/UnrealRider/Verse-Bites/tree/main/Lesson-Bites)

### 第1步 : 安装 VS Code
- 在开启你的 UEFN 之前，请先安装完成 VS Code (Visual Studio Code)
- [请由 Visual Studio Code 官方网址下载安装](https://code.visualstudio.com/) 

### 第2步 : 开启你的 UEFN 项目
- 开启或新建你学习用的 UEFN 项目
- 推荐使用 Blank 模板，这个模板作为学习或正式项目的测试环境是最佳的选择，干净极简的环境有助于专注力的提高与加载的效率.    
  - <img width="1469" height="879" alt="5a98389c-12d6-4e05-9e31-31c55356e098" src="https://github.com/user-attachments/assets/517c3530-8e93-42c6-8eb2-548a6f7a73b5" />

### 第3步 : 构建 Verse 目录
- 3.1 在 UEFN 上方的工具栏点击 `Verse > Open Project in Vs Code`, 这将开启 VS Code，并直接进入项目的工作文件夹。
  - <img width="976" height="294" alt="image" src="https://github.com/user-attachments/assets/0939d3dd-6f33-4d93-a4e2-66e4480ac6b4" />

- 3.2 提示 : VS Code 的项目文件夹与 UEFN Content Browser 的项目文件夹是相同的，所以不论你从哪个软件中建立文件夹，2边都是同步的。
  - <img width="1065" height="440" alt="0b0bbedd-0b9e-4e09-b2ef-4ec68480482f" src="https://github.com/user-attachments/assets/315a4b9d-0774-414d-9c8d-8288a0bb4f18" />

- 3.3 开始建构文件目录，我们直接在 VS Code 中建立 Verse 的文件，请按照下面的结构建立文件夹:
  ```
  - Verse
    - VerseBites
      - LessonBites
  ```   
- 文件夹结构说 :
  - Verse 文件夹是你统一存放 Verse 代码的主要位置
  - VerseBites 文件夹存放所有 Verse-Bites 资源的位置
  - LessonBites 文件夹存放 LessonBites 学习课程的相关资源的位置

  -  VS Code 与 UEFN Content Browser 对照:   
    - <img width="885" height="375" alt="0123c254-0ac0-4f11-bdcb-6c125b72d816" src="https://github.com/user-attachments/assets/97435a06-f3f9-46fb-a658-2accba765df1" />

- 3.4 提醒 : 由 .verse 文件所产生的 Device，会生成在与 .verse 相同的文件夹中，而 .verse 文件在不同的文件夹会有上下层级的关系，这会影响到代码之间的调用方法，所以在项目开始前做好文件夹目录的结构管理是很重要的

### 第4步 : 复制 Lesson-Bites 的代码
- 4.1 在 VS Code 进行操作，在 LessonBites 文件夹下新建一个 Lesson-Bites_L1.verse 文件
- 4.2 将 Github 上的 [Lesson-Bites_L1.verse](https://github.com/UnrealRider/Verse-Bites/blob/main/Lesson-Bites/Lesson-Bites_L1.verse) 代码内容复制到你的 Lesson-Bites_L1.verse 文件内, 然后存档    
  - <img width="1890" height="743" alt="7b50bf8c-455d-4adc-a7e5-0f136c43cc6b" src="https://github.com/user-attachments/assets/a371a7da-b0ef-414f-97e1-46fc8000f9a6" />
- 4.3 Build Lesson-Bites_L1.verse ，按下 VS Code 右上的 Build 按钮，或 UEFN 中的 Verse Build 按钮
  - <img width="1273" height="409" alt="014d7298-52da-4418-bf69-5611bd8dd644" src="https://github.com/user-attachments/assets/8b6d9a4b-69bc-4ead-b147-a11193b349de" />
- 4.4 Build 成功将会显示成功信息
  - <img width="1273" height="946" alt="50ac188f-d48a-4f17-957a-958150a54328" src="https://github.com/user-attachments/assets/3226b258-50aa-4536-a9c3-99127f222b64" />
- 4.5 Build 成功后 UEFN 的 LessonBites 文件夹下将会产生多个 Verse Device
- 提示 : 如果你在 4.3 步骤后编译失败，请到 Discord 上提供错误的信息

### 第5步 : 跟着 Lesson-Bites_L1.verse 代码的注释内容与教学视频开始学习
- 进入 [Lesson-Bites Roadmap.md](https://github.com/UnrealRider/Verse-Bites/blob/main/Lesson-Bites/Lesson-Bites%20Roadmap.md), 选择对应的课程视频，开始学习
- 第一个课程会说明如何使用 Lesson-Bites_L1.verse，当你已经熟悉使用了，后续的课程内容你可以直接参照代码中的注释直接学习
